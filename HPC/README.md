# Words-in-Progress-NLP-for-Language-Learning

Set up to be run as a SLURM job on the UVA system. To run:
1) log into Rivanna Server
2) add job_script.sh and translation_model.py to the same folder in server
3) edit job_script.sh and specify values for huggingface token and google api key
4) submit as job with sbatch job.sh
